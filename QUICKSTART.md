# 빠른 시작 가이드

## 1분 안에 시작하기

### 1단계: 설치
```bash
npm install
```

### 2단계: 백엔드 서버 실행
다른 터미널에서:
```bash
# 백엔드 디렉토리로 이동
cd /path/to/backend

# FastAPI 서버 실행
python -m uvicorn main:app --reload
```

서버가 `http://localhost:8000`에서 실행되어야 합니다.

### 3단계: 프론트엔드 실행
```bash
npm run dev
```

### 4단계: 브라우저에서 열기
브라우저에서 `http://localhost:5173` 접속

---

## 주요 기능 사용법

### 🎯 이미지 예측 (메인 페이지)

#### 방법 1: 파일 업로드
1. "이미지 선택" 버튼 클릭
2. 숫자 이미지 선택
3. 자동으로 예측 결과 표시

#### 방법 2: 직접 그리기
1. 캔버스에 숫자 그리기
2. "인식하기" 버튼 클릭
3. 예측 결과 확인

---

### 🧠 모델 학습 (/train 페이지)

#### 간단한 학습
1. 네비게이션에서 "모델 학습" 클릭
2. 기본값 그대로 "학습 시작" 클릭
3. 실시간 진행상황 확인
   - 에포크별 손실값
   - 정확도 변화
   - 진행 바
   - **실시간 그래프**: 손실/정확도 라인 차트

#### 커스텀 학습
1. 학습률(η) 입력: 예) 0.01
2. 람다(λ) 입력: 예) 0.0001
3. "학습 후 모델 저장" 체크
4. "학습 시작" 클릭

#### 학습 중지
- "학습 중지" 버튼 클릭

#### 그래프 보기
학습이 진행되면 자동으로 두 개의 그래프가 표시됩니다:
- **손실 그래프** (빨간색): 에포크별 손실값 감소 추이
- **정확도 그래프** (녹색): 에포크별 정확도 향상 추이

---

### 🧪 모델 테스트

1. "모델 학습" 페이지의 "모델 테스트" 카드
2. "테스트 실행" 버튼 클릭
3. 정확도 확인
   - 퍼센트 표시
   - 정답 개수 / 전체

---

### ⚙️ 하이퍼파라미터 최적화

**⚠️ 주의: 시간이 오래 걸립니다!**

#### 테스트용 (빠름)
```
학습률 후보: 0.01, 0.02
람다 후보: 0.0001, 0.001
반복 횟수: 2
```

#### 실전용 (느림)
```
학습률 후보: 0.009, 0.01, 0.02, 0.03, 0.05, 0.06
람다 후보: 0.00001, 0.0001, 0.001
반복 횟수: 5
```

**사용법:**
1. 하이퍼파라미터 최적화 카드에서 값 입력
2. "최적화 시작" 클릭
3. 실시간 진행상황 확인
   - 현재 조합
   - 현재 최고 성능
   - **막대 그래프**: 각 조합의 정확도 비교 (최고 성능은 녹색으로 표시)
4. 완료 후 전체 결과 테이블 확인

## 문제 해결

### "WebSocket 연결 끊김"
→ 백엔드 서버가 실행 중인지 확인

### "모델이 현재 학습 중입니다"
→ 학습 완료 대기 또는 "학습 중지" 클릭

### "예측 실패"
→ 백엔드 서버 상태 확인 (`http://localhost:8000/docs`)

### 화면이 이상하게 보임
→ 브라우저 캐시 삭제 (Cmd/Ctrl + Shift + R)
