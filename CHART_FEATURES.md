# 📊 차트 기능 가이드

## 개요

학습 진행상황과 최적화 결과를 실시간 그래프로 시각화합니다.

---

## 🎯 학습 그래프

### 위치
`/train` 페이지 → 모델 학습 카드 하단

### 표시 조건
- 학습이 시작되고 최소 2개 이상의 에포크 데이터가 있을 때 자동 표시

### 제공되는 그래프

#### 1. 손실 그래프 (Loss Chart)
- **색상**: 빨간색 (#ef4444)
- **내용**: 에포크별 손실값 변화
- **Y축**: 손실값 (자동 스케일)
- **X축**: 에포크 번호
- **특징**:
  - 라인 차트로 표시
  - 각 데이터 포인트에 점 표시
  - 그리드 라인으로 값 읽기 쉬움
  - 손실이 감소하는 추세를 시각적으로 확인

#### 2. 정확도 그래프 (Accuracy Chart)
- **색상**: 녹색 (#10b981)
- **내용**: 에포크별 정확도(%) 변화
- **Y축**: 정확도 퍼센트 (0-100%)
- **X축**: 에포크 번호
- **특징**:
  - 라인 차트로 표시
  - 각 데이터 포인트에 점 표시
  - 그리드 라인으로 값 읽기 쉬움
  - 정확도가 향상되는 추세를 시각적으로 확인

### 사용 예시

```
학습 시작 → 실시간 그래프 업데이트

Epoch 1:  Loss 0.523, Acc 85.2%  ●━━━━━━━━━
Epoch 5:  Loss 0.312, Acc 91.3%  ●━━━●━━━━━
Epoch 10: Loss 0.201, Acc 94.1%  ●━━━●━━━●━
Epoch 30: Loss 0.089, Acc 97.2%  ●━━━●━━━●━●
```

## ⚙️ 최적화 그래프

### 위치
`/train` 페이지 → 하이퍼파라미터 최적화 카드

### 표시 조건
- 최적화가 시작되고 결과 데이터가 있을 때 자동 표시

### 제공되는 그래프

#### 조합별 정확도 막대 그래프
- **타입**: 막대 그래프 (Bar Chart)
- **색상**:
  - 일반 조합: 보라색 (#667eea, 투명도 70%)
  - 최고 성능: 녹색 (#10b981, 투명도 100%)
- **내용**: 각 하이퍼파라미터 조합의 정확도
- **Y축**: 정확도 퍼센트
- **X축**: 조합 번호
- **특징**:
  - 최고 성능 조합은 녹색으로 강조
  - 최고 성능 위에 정확한 수치 표시
  - 범례로 일반/최고 구분
  - 한눈에 성능 비교 가능

### 사용 예시

```
최적화 진행 중...

조합 1: η=0.009, λ=0.00001 → 94.2%  ▓▓▓▓▓▓▓▓▓░
조합 2: η=0.010, λ=0.00001 → 95.5%  ▓▓▓▓▓▓▓▓▓▓ ⭐ (최고)
조합 3: η=0.020, λ=0.00001 → 93.8%  ▓▓▓▓▓▓▓▓░░
...
```

## 🎨 차트 디자인

### 공통 특징
- **SVG 기반**: 확대/축소해도 깨끗한 벡터 그래픽
- **반응형**: 컨테이너 크기에 맞춰 자동 조절
- **그리드**: 값을 쉽게 읽을 수 있는 그리드 라인
- **축 레이블**: 명확한 Y축 값과 X축 레이블
- **부드러운 애니메이션**: 데이터 업데이트 시 자연스러운 전환

### 색상 팔레트
- 손실: `#ef4444` (빨간색)
- 정확도: `#10b981` (녹색)
- 일반: `#667eea` (보라색)
- 그리드: `#e5e7eb` (연한 회색)
- 레이블: `#6b7280` (회색)

## 💡 활용 팁

### 학습 그래프
1. **과적합 감지**
   - 손실이 다시 증가하는 구간 확인
   - 정확도가 정체되는 지점 확인

2. **학습률 조정**
   - 손실이 너무 천천히 감소 → 학습률 증가 고려
   - 손실이 불규칙하게 변동 → 학습률 감소 고려

3. **학습 진행 예측**
   - 초반 에포크의 추세로 전체 학습 결과 예측
   - 조기 종료 판단 가능

### 최적화 그래프
1. **최적 조합 발견**
   - 녹색 막대가 최고 성능 조합
   - 비슷한 성능의 조합들 비교

2. **파라미터 경향 파악**
   - 어떤 학습률/람다 범위가 좋은지 확인
   - 테이블과 함께 보면 더 자세한 분석 가능

3. **추가 최적화 계획**
   - 최고 성능 주변 값들로 세밀하게 재탐색
   - 더 넓은 범위 또는 좁은 범위 시도 결정
