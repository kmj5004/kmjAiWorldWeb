# Neural Network API - 사용 가이드

이 프로젝트는 MNIST 손글씨 숫자를 예측하는 FastAPI 기반 API와, 이미지를 업로드하고 예측을 시도하는 Svelte 프런트엔드로 구성됩니다.

## 이미지 형식

- 모든 네트워크는 28x28 픽셀의 회색조 이미지를 입력으로 받습니다.
- 총 784개의 픽셀 값 (28 × 28 = 784)
- 각 값은 0.0 (흰색/배경)에서 1.0 (검은색/전경) 사이
- JSON 배열 형식으로 전달

### 프런트엔드 이미지 처리 과정

1. 사용자가 이미지 파일을 업로드
2. 28x28 픽셀로 다운스케일
   - 흰색 배경 캔버스에 이미지를 그림 (투명 배경 방지)
3. RGB를 grayscale로 변환: `(R + G + B) / 3`
4. 정규화: `1.0 - (grayscale / 255.0)`
   - 흰색 배경 → 0.0
   - 검은색 전경 → 1.0
5. 784개 픽셀 배열을 API로 전송


## 라이선스

이 프로젝트는 교육 목적으로 사용됩니다.
